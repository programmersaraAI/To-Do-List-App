# To-Do-List-App

## Description

A simple To-Do List application that allows users to add, complete, and delete tasks, enhancing productivity and task management.

## Features

- **User Authentication**: Built using Firebase (or you can use local storage for simplicity).
- **CRUD Operations**: Create, Read, Update, and Delete tasks effortlessly.
- **MVVM Architecture**: Clear separation of business logic from UI.
- **Local Notifications**: Remind users of tasks via local notifications.

## Technologies

- **Language**: Swift
- **Framework**: UIKit
- **Data Persistence**: Core Data/Realm (optional integration with Firebase)
- **Architecture**: MVVM
- **Networking**: RESTful APIs (optional)
